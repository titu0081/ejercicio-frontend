# EjercicioFrontend

Versión de angular utilizada 20.0.4

## Iniciar el proyecto

    1. Instalar todas las dependencias (node modules) con npm install
    2. Iniciar el proyecto con el comando: ng serve  o  npm start

## Ejecutar las pruebas con jest

Comandos a utilizar: npm test

## Ruta configurable

    La ruta para la conexión con el backend se encuentra en la carpeta:
    - environments/environment.ts
